# 你该换工作，还是换活法

一个无需后端的中文互动测评站。通过 18 个近期生活与工作情境，比较“工作错配”“生活失衡”和“判断透支”三组信号，给出一个应优先验证的改变方向。

## 页面结构

- 首页：说明测试目标，并引导用户开始判断。
- 测试：18 道单选题，每个方向 6 题，支持返回修改和退出后继续。
- 分析：短暂的路线校准过渡。
- 结果：展示主要方向、全部归一化分数、解释和 7 天验证行动。

答案只保留在当前页面状态中，刷新页面会清空。结果用于自我梳理，不是离职概率或专业建议。

## 本地查看

```powershell
py -m http.server 4173
```

然后访问 `http://127.0.0.1:4173/`。

## 构建与验证

```powershell
node --check app.js
npm run build
```

构建结果位于：

- `dist/public/`：可直接上传到静态托管平台的前端文件。
- `dist/server/index.js`：内嵌同一套静态资源的 Worker 版本。
