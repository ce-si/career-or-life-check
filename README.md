# 你适合年上，还是年下？

一套无需后端的中文互动测评与小红书商品素材。测评通过 18 个具体关系情境，比较“稳定承接感”“鲜活带动感”和“平等同频感”三组需求信号，帮助成年用户观察自己更舒服的亲密关系节奏。

## 页面结构

- 首页：说明测评目标，并引导用户开始校准。
- 测试：18 道单选题，每个维度 6 题，支持返回修改和退出后继续。
- 分析：短暂的关系信号校准过渡。
- 结果：展示主要倾向、三组归一化信号、解释和 3 条关系观察提示。

答案只保留在当前页面状态中，刷新页面会清空。结果只用于自我观察，不是心理测评、匹配概率或择偶结论。

## 本地查看

```powershell
py -m http.server 4173
```

然后访问 `http://127.0.0.1:4173/`。

## 构建与验证

```powershell
node --check app.js
npm run build
```

构建结果位于：

- `dist/public/`：可直接上传到静态托管平台的前端文件。
- `dist/server/index.js`：内嵌同一套静态资源的 Worker 版本。
